# Twitter-to-Telegram-bot
Simply send telegram message while new tweet sent.
## Usage
Please add config.json contains
```
{
    "TelegramBotToken": "...",
    "consumer_key": "...",
    "consumer_secret": "...",
    "timeout_ms": "60000",
    "access_token": "...",
    "access_token_secret": "...",
    "screen_name": "..."
}
```
Screen_name is twitter user name which you want to forward.
